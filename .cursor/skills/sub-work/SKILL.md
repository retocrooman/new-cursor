# Sub Work

## Instructions

- 必ずサブエージェントに作業を任せる。
- 必ず worktree で実行する。
- 実行前にベースブランチをユーザーに確認する。
- 実行前に作業ブランチをユーザーに確認する。
- 実装前に不明点があれば、必ず親エージェントで仕様確認する。
