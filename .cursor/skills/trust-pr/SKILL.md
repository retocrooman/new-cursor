# Trust PR

## Instructions

- PR は `--auto-merge` を付けて作成する。
- PR 作成前に現在ブランチと push 状態を確認する。
- 未push の場合は、PR 作成前に push する。
- PR 作成後は、PR URL だけ簡潔に伝える。
