# UI

- 画面構成
  - 左：タスクリストを置く
  - 中央：タスク詳細・ダッシュボードを置く
  - 右：司令官とのチャットを置く
- タスクリスト
  - レポジトリ名を必ず表示する
  - 実行時間を表示する
  - 消費トークン数を表示する
  - 最後のイベントを表示する
  - 状態をアイコンで表示する
- ダッシュボード
  - 重要な意思決定リストを表示する
  - イベントヒストリーを表示する
  - 作業ファイルの一覧と要約を表示する
  - コードの変更量を表示する
  - ブランチ名を表示する
  - レポジトリ名を表示する
- 司令官チャット
  - 起票時は、タスクタイトルを必ずユーザーに確認する
  - 起票時は、ベースブランチを必ずユーザーに確認する
  - 起票時は、ブランチ名を必ずユーザーに確認する
  - 起票時は、背景を必ずユーザーに確認する
  - 起票時は、検証項目を必ずユーザーに確認する
